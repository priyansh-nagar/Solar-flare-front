A = 10⁻⁸
B = 10⁻⁷
C = 10⁻⁶
M = 10⁻⁵
X = 10⁻⁴

Since:

3.11 × 10⁻⁵

falls in the M-class range, showing M is correct.

You show:

WARNING
Current Class = M

If the Sun is currently in M-class activity, a warning state makes sense.
